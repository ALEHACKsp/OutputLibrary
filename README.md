# C++ Console Output Library

<h3> Features:
<li> Print (const chars, strings, numbers, etc...)
<li> Newline
<li> Pause Console
<li> Print Exclamations, Pluses, Dollar Signs, Example: "[+] Hi" or "[!] Failed..."
<li> Generate Random Strings
